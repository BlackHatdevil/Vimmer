# Vimmer——专为Linuxer设计的vim配置，精简优雅
强大的配置文件，让你的vim成为脚本的IDE，特别适合Web开发者使用

####以下语言均适用，按体验等级划分：
1. php
2. HTML CSS JS
3. Jquery
4. C／C++
5. Python
6. markdown

###安装方法：
```
cd ~
git clone https://github.com/BlackHatdevil/Vimmer.git
cd ~/Vimmer
mv .vim .vimrc ~
rm -rf ~/Vimmer
```
就这么简单！打开vim看看👀不一样的变化！

###操作方法：

- html：ctrl ＋ O //自动生成闭合标签，参考emmet的使用
- css：自动颜色显示
- 保存：F5
- 格式化（缕顺）代码：F12
- 脚本跟踪查错并保存并格式化代码：F6
- 去除所有空行：F12
- 召唤树状目录：F3
- 召唤taglist：F9
- 自动补全：已自动化，C－X C－O全能补全
- 粘贴系统剪切板内容：C－V
